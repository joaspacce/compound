# Compound Engineering Plugin - Presentation

A concise overview of the [Compound Engineering Plugin](https://github.com/EveryInc/compound-engineering-plugin) by Every Inc.

## How to View

Open in your browser:

```bash
open presentation/index.html
```

Or double-click `index.html` in your file manager.

## Controls

- **Arrow keys** or **Space** to navigate slides
- **Esc** to see slide overview
- **F** for fullscreen
- **S** for speaker notes view
