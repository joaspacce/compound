# Compound Engineering Plugin - Presentation

Slide deck about the [Compound Engineering Plugin](https://github.com/EveryInc/compound-engineering-plugin) by [Every Inc](https://github.com/EveryInc), prepared for **Kelsus Camp 2026**.

## View

Open `presentation/index.html` in any browser.

## Features

- Dark/light mode toggle (default dark)
- Multilingual: English, Spanish, Portuguese
- Kelsus SVG logo with animated cursor
- Preferences saved in localStorage

## Built With

- [Reveal.js 5](https://revealjs.com/) (CDN, no build step)
